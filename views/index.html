Oxford Dictionary API

Get word definitions

Usage: set the query parameter define equal to the word you want the meaning of.

Eg.: /?define=goodbye, will return:

  
  {
    "word": "goodbye",
    "phonetic": "/ɡʊdˈbʌɪ/",
    "origin": "Late 16th century contraction of God be with you!, with good substituted on the pattern of phrases such as good morning.",
    "meaning": {
        "exclamation": [
            {
                "definition": "Used to express good wishes when parting or at the end of a conversation.",
                "synonyms": [
                    "farewell",
                    "adieu"
                ]
            }
        ],
        "noun": [
            {
                "definition": "An instance of saying ‘goodbye’; a parting.",
                "example": "we said our goodbyes and set off",
                "synonyms": [
                    "goodbye",
                    "farewell"
                ]
            }
        ]
    }
  }



Language support


Optionally you can send an additional parameter lang set equal to the language the word comes from.

For language other than English, this API scrapes from Google (as oxford doesn't provide dictionary of languages other than English)

Eg.: /?define=prarabdh&lang=hi, will give you definition of prarabdh:



  {
    "word": "प्रारब्ध",
    "meaning": {
        "विशेषण": [
            {
                "definition": "आरंभ किया हुआ।"
            }
        ],
        "पुल्लिंग": [
            {
                "definition": "भाग्य, तकदीर।"
            },
            {
                "definition": "पूर्व जन्म के कार्य।"
            }
        ]
    }
  }
